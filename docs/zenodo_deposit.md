# Zenodo Deposit

Before creating the Zenodo archive:

1. Run `git lfs pull` so large files are present locally.
2. Confirm `data_manifest.csv` has checksums for every included raw data file.
3. Decide whether generated `results/shiny_data` RDS files are included or
   regenerated by users.
4. Confirm the archive contains real data files rather than Git LFS pointer
   files.
5. Record the final GitHub release tag and Zenodo DOI here.
